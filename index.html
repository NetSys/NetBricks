NetBricks is a safe and fast framework for rapid development of network functions for use in NFV written using Rust and DPDK. NetBricks aims to make it faster to develop new NFs, while also providing faster mechanisms for safely chaining NFs. The code is still work in progress and is changing, but Pull Requests and suggestions are welcome.

Our OSDI 2016 paper describing NetBricks is available here.
